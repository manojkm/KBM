# KBM Resorts - Email Inbox

A modern, responsive email inbox interface built with Bootstrap 5 and Vite.

## Overview

This project provides a comprehensive email management system for KBM Resorts, featuring an intuitive inbox interface with advanced filtering, search capabilities, and conversation management.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Email Management**: View, organize, and manage emails with an intuitive interface
- **Advanced Filtering**: Filter emails by context, sub-context, and role using dropdown filters
- **Search Functionality**: Search emails with support for multiple search criteria
- **Conversation History**: View email and SMS conversation threads in a chat-like interface
- **Message Recipients**: Manage and select message recipients with a sortable table interface
- **Email Composition**: Compose and reply to emails with attachment support
- **Modern UI**: Clean, modern interface built with Bootstrap 5

## Tech Stack

- **Frontend Framework**: Bootstrap 5.3
- **Build Tool**: Vite
- **CSS Preprocessor**: Sass
- **Icons**: Bootstrap Icons
- **Fonts**: Google Fonts (Inter)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd KBM
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and navigate to `http://localhost:5173`
